# 使用 RestSharp 进行服务器端开发的完整指南

下面是使用 RestSharp 进行服务器端开发的详细步骤，从安装 VS Code 开始：

## 第一部分：环境设置

### 1. 安装 Visual Studio Code

1. 访问 VS Code 官网 (https://code.visualstudio.com/)
2. 下载适合您操作系统的版本 (Windows/macOS/Linux)
3. 运行安装程序并按照提示完成安装
4. 启动 VS Code

### 2. 安装必要的 VS Code 扩展

1. 打开 VS Code
2. 点击左侧扩展图标或按 `Ctrl+Shift+X`
3. 搜索并安装以下扩展：
    - C# (由 Microsoft 提供)
    - .NET Core Test Explorer
    - REST Client (可选，用于测试 API)

### 3. 安装 .NET SDK

1. 访问 .NET 下载页面 (https://dotnet.microsoft.com/download)
2. 下载并安装最新的 .NET SDK
3. 安装完成后，打开终端并运行 `dotnet --version` 确认安装成功

## 第二部分：创建测试项目

### 1. 创建新的 Web API 项目

1. 打开终端，导航到您想创建项目的目录
2. 运行以下命令创建 Web API 项目：

```bash
dotnet new webapi -n RestSharpServerDemo
cd RestSharpServerDemo
```

### 2. 创建测试项目

1. 在项目目录中，创建一个测试项目：

```bash
dotnet new xunit -n RestSharpServerDemo.Tests
cd RestSharpServerDemo.Tests
```

1. 添加对主项目的引用：

```bash
dotnet add reference ../RestSharpServerDemo/RestSharpServerDemo.csproj
```

### 3. 添加 RestSharp 包

1. 回到测试项目目录，添加 RestSharp 包：

```bash
dotnet add package RestSharp
```

1. 同时添加其他可能需要的包：

```bash
dotnet add package Microsoft.AspNetCore.TestHost
dotnet add package FluentAssertions
```

### 4. 打开项目

运行以下命令在 VS Code 中打开项目：

```bash
cd ..
code .
```

## 第三部分：实现基本 API 端点

### 1. 创建模型类

在主项目中创建一个简单的模型类：

1. 在 `RestSharpServerDemo` 项目中创建 `Models` 文件夹
2. 在该文件夹中添加 `Product.cs` 文件：

```csharp
namespace RestSharpServerDemo.Models
{
    public class Product
    {
        public int Id { get; set; }
        public string Name { get; set; }
        public decimal Price { get; set; }
        public string Description { get; set; }
    }
}
```

### 2. 创建控制器

在 `Controllers` 文件夹中创建 `ProductsController.cs`：

```csharp
using System.Collections.Generic;
using Microsoft.AspNetCore.Mvc;
using RestSharpServerDemo.Models;

namespace RestSharpServerDemo.Controllers
{
    [ApiController]
    [Route("api/[controller]")]
    public class ProductsController : ControllerBase
    {
        private static List<Product> _products = new List<Product>
        {
            new Product { Id = 1, Name = "笔记本电脑", Price = 5999.99m, Description = "高性能笔记本电脑" },
            new Product { Id = 2, Name = "智能手机", Price = 3999.99m, Description = "最新款智能手机" },
            new Product { Id = 3, Name = "平板电脑", Price = 2999.99m, Description = "超薄平板电脑" }
        };

        // GET: api/products
        [HttpGet]
        public ActionResult<IEnumerable<Product>> GetProducts()
        {
            return _products;
        }

        // GET: api/products/1
        [HttpGet("{id}")]
        public ActionResult<Product> GetProduct(int id)
        {
            var product = _products.Find(p => p.Id == id);
            if (product == null)
            {
                return NotFound();
            }
            return product;
        }

        // POST: api/products
        [HttpPost]
        public ActionResult<Product> CreateProduct(Product product)
        {
            product.Id = _products.Count + 1;
            _products.Add(product);
            return CreatedAtAction(nameof(GetProduct), new { id = product.Id }, product);
        }

        // PUT: api/products/1
        [HttpPut("{id}")]
        public IActionResult UpdateProduct(int id, Product product)
        {
            var existingProduct = _products.Find(p => p.Id == id);
            if (existingProduct == null)
            {
                return NotFound();
            }

            existingProduct.Name = product.Name;
            existingProduct.Price = product.Price;
            existingProduct.Description = product.Description;

            return NoContent();
        }

        // DELETE: api/products/1
        [HttpDelete("{id}")]
        public IActionResult DeleteProduct(int id)
        {
            var product = _products.Find(p => p.Id == id);
            if (product == null)
            {
                return NotFound();
            }

            _products.Remove(product);
            return NoContent();
        }
    }
}
```

## 第四部分：使用 RestSharp 编写测试

在测试项目中创建 `ProductApiTests.cs` 文件：

```csharp
using System;
using System.Collections.Generic;
using System.Net;
using System.Threading.Tasks;
using FluentAssertions;
using Microsoft.AspNetCore.Hosting;
using Microsoft.AspNetCore.TestHost;
using RestSharp;
using RestSharpServerDemo;
using RestSharpServerDemo.Models;
using Xunit;
using Newtonsoft.Json;

namespace RestSharpServerDemo.Tests
{
    public class ProductApiTests : IDisposable
    {
        private readonly TestServer _server;
        private readonly RestClient _client;

        public ProductApiTests()
        {
            // 创建测试服务器
            _server = new TestServer(new WebHostBuilder()
                .UseStartup<Startup>());
            
            // 初始化 RestSharp 客户端
            _client = new RestClient(_server.BaseAddress.ToString());
        }

        public void Dispose()
        {
            _server.Dispose();
        }

        [Fact]
        public async Task GetProducts_ReturnsAllProducts()
        {
            // 安排
            var request = new RestRequest("api/products", Method.GET);

            // 执行
            var response = await _client.ExecuteAsync<List<Product>>(request);

            // 断言
            response.StatusCode.Should().Be(HttpStatusCode.OK);
            response.Data.Should().NotBeNull();
            response.Data.Count.Should().BeGreaterThan(0);
        }

        [Fact]
        public async Task GetProduct_WithValidId_ReturnsProduct()
        {
            // 安排
            var request = new RestRequest("api/products/1", Method.GET);

            // 执行
            var response = await _client.ExecuteAsync<Product>(request);

            // 断言
            response.StatusCode.Should().Be(HttpStatusCode.OK);
            response.Data.Should().NotBeNull();
            response.Data.Id.Should().Be(1);
        }

        [Fact]
        public async Task GetProduct_WithInvalidId_ReturnsNotFound()
        {
            // 安排
            var request = new RestRequest("api/products/999", Method.GET);

            // 执行
            var response = await _client.ExecuteAsync(request);

            // 断言
            response.StatusCode.Should().Be(HttpStatusCode.NotFound);
        }

        [Fact]
        public async Task CreateProduct_AddsNewProduct()
        {
            // 安排
            var newProduct = new Product
            {
                Name = "测试产品",
                Price = 199.99m,
                Description = "这是一个测试产品"
            };

            var request = new RestRequest("api/products", Method.POST);
            request.AddJsonBody(newProduct);

            // 执行
            var response = await _client.ExecuteAsync<Product>(request);

            // 断言
            response.StatusCode.Should().Be(HttpStatusCode.Created);
            response.Data.Should().NotBeNull();
            response.Data.Name.Should().Be(newProduct.Name);
            response.Data.Id.Should().BeGreaterThan(0);
        }

        [Fact]
        public async Task UpdateProduct_WithValidId_UpdatesProduct()
        {
            // 安排 - 首先获取现有产品
            var getRequest = new RestRequest("api/products/1", Method.GET);
            var getResponse = await _client.ExecuteAsync<Product>(getRequest);
            var existingProduct = getResponse.Data;

            // 修改产品
            existingProduct.Name = "更新的产品名称";
            existingProduct.Price = 1299.99m;

            var updateRequest = new RestRequest("api/products/1", Method.PUT);
            updateRequest.AddJsonBody(existingProduct);

            // 执行
            var updateResponse = await _client.ExecuteAsync(updateRequest);

            // 断言
            updateResponse.StatusCode.Should().Be(HttpStatusCode.NoContent);

            // 验证更新是否成功
            var verifyRequest = new RestRequest("api/products/1", Method.GET);
            var verifyResponse = await _client.ExecuteAsync<Product>(verifyRequest);
            
            verifyResponse.Data.Name.Should().Be("更新的产品名称");
            verifyResponse.Data.Price.Should().Be(1299.99m);
        }

        [Fact]
        public async Task DeleteProduct_WithValidId_RemovesProduct()
        {
            // 安排
            var request = new RestRequest("api/products/3", Method.DELETE);

            // 执行
            var response = await _client.ExecuteAsync(request);

            // 断言
            response.StatusCode.Should().Be(HttpStatusCode.NoContent);

            // 验证删除是否成功
            var verifyRequest = new RestRequest("api/products/3", Method.GET);
            var verifyResponse = await _client.ExecuteAsync(verifyRequest);
            
            verifyResponse.StatusCode.Should().Be(HttpStatusCode.NotFound);
        }
    }
}

```

## 第五部分：RestSharp 的高级用法

### 1. 设置请求头和认证

```csharp
// 添加请求头
var request = new RestRequest("api/secured-resource", Method.GET);
request.AddHeader("Content-Type", "application/json");
request.AddHeader("Authorization", "Bearer your-token-here");

// 基本认证
request.Authenticator = new HttpBasicAuthenticator("username", "password");

// OAuth2认证
request.Authenticator = new OAuth2AuthorizationRequestHeaderAuthenticator(
    "your-access-token", 
    "Bearer");
```

### 2. 处理文件上传

```csharp
var request = new RestRequest("api/upload", Method.POST);
request.AddFile("file", "/path/to/file.pdf", "application/pdf");

// 或从字节数组上传
byte[] fileBytes = System.IO.File.ReadAllBytes("/path/to/file.jpg");
request.AddFile("file", fileBytes, "photo.jpg", "image/jpeg");
```

### 3. 处理复杂参数

```csharp
// URL查询参数
var request = new RestRequest("api/products", Method.GET);
request.AddQueryParameter("category", "electronics");
request.AddQueryParameter("minPrice", "1000");

// URL路径参数
var request = new RestRequest("api/products/{id}/reviews/{reviewId}", Method.GET);
request.AddUrlSegment("id", "42");
request.AddUrlSegment("reviewId", "123");
```

### 4. 异步操作

```csharp
public async Task<List<Product>> GetProductsAsync()
{
    var request = new RestRequest("api/products", Method.GET);
    var response = await _client.ExecuteAsync<List<Product>>(request);
    
    if (response.IsSuccessful)
    {
        return response.Data;
    }
    
    throw new Exception($"API错误: {response.ErrorMessage}");
}
```

### 5. 错误处理

```csharp
var request = new RestRequest("api/products", Method.GET);
var response = await _client.ExecuteAsync<List<Product>>(request);

if (!response.IsSuccessful)
{
    if (response.StatusCode == HttpStatusCode.NotFound)
    {
        // 处理404错误
    }
    else if (response.StatusCode == HttpStatusCode.Unauthorized)
    {
        // 处理401错误
    }
    else
    {
        // 处理其他错误
        Console.WriteLine($"错误: {response.ErrorMessage}");
        // 检查响应内容获取更多详情
        Console.WriteLine($"响应内容: {response.Content}");
    }
}
```

### 6. 自定义序列化设置

```csharp
var client = new RestClient(_server.BaseAddress.ToString());
client.UseNewtonsoftJson(new JsonSerializerSettings
{
    NullValueHandling = NullValueHandling.Ignore,
    DateTimeZoneHandling = DateTimeZoneHandling.Utc
});
```

```bash
cd RestSharpServerDemo.Tests
dotnet test
```

## 第七部分：部署和维护

### 1. 发布应用程序

```bash
dotnet publish -c Release
```

### 2. Docker 支持（可选）

1. 在项目根目录创建 `Dockerfile`：

```dockerfile
FROM mcr.microsoft.com/dotnet/aspnet:5.0 AS base
WORKDIR /app
EXPOSE 80
EXPOSE 443

FROM mcr.microsoft.com/dotnet/sdk:5.0 AS build
WORKDIR /src
COPY ["RestSharpServerDemo/RestSharpServerDemo.csproj", "RestSharpServerDemo/"]
RUN dotnet restore "RestSharpServerDemo/RestSharpServerDemo.csproj"
COPY . .
WORKDIR "/src/RestSharpServerDemo"
RUN dotnet build "RestSharpServerDemo.csproj" -c Release -o /app/build

FROM build AS publish
RUN dotnet publish "RestSharpServerDemo.csproj" -c Release -o /app/publish

FROM base AS final
WORKDIR /app
COPY --from=publish /app/publish .
ENTRYPOINT ["dotnet", "RestSharpServerDemo.dll"]
```

1. 构建并运行 Docker 容器：

```bash
docker build -t restsharp-server-demo .
docker run -p 8080:80 restsharp-server-demo
```

 
在 **React 项目**中，条件渲染（**Rendering Content Conditionally**）是很常见的需求，比如登录状态不同显示不同的内容、按钮点击后隐藏/显示某些元素等等。

React 提供了几种实现方式：

---

## 1. 使用 `if` 语句

最直观的方式是在渲染函数里用 `if` 判断：

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  } else {
    return <h1>Please sign in.</h1>;
  }
}

// 使用
<Greeting isLoggedIn={true} />   // 输出 Welcome back!
<Greeting isLoggedIn={false} />  // 输出 Please sign in.
```

---

## 2. 使用三元运算符 `? :`

适合简单的场景，直接写在 JSX 里：

```jsx
function UserStatus({ isOnline }) {
  return (
    <div>
      {isOnline ? <span style={{color: "green"}}>🟢 Online</span> 
                : <span style={{color: "red"}}>🔴 Offline</span>}
    </div>
  );
}

// 使用
<UserStatus isOnline={true} />
```

---

## 3. 使用逻辑与 `&&`

当只需要在满足条件时渲染某个内容，可以用 `&&`：

```jsx
function Notification({ message }) {
  return (
    <div>
      <h2>My App</h2>
      {/* 只有在 message 存在时才显示 */}
      {message && <p>{message}</p>}
    </div>
  );
}

// 使用
<Notification message="You have 3 new messages!" />
<Notification message="" />   // 什么也不显示
```

---

## 4. 多条件渲染（switch / 映射）

当条件比较多时，可以用 `switch` 或者一个对象映射：

```jsx
function Status({ status }) {
  switch (status) {
    case "loading":
      return <p>Loading...</p>;
    case "error":
      return <p style={{color: "red"}}>Error occurred!</p>;
    case "success":
      return <p>✅ Success!</p>;
    default:
      return <p>Unknown status</p>;
  }
}

// 使用
<Status status="loading" />
<Status status="success" />
```

---

## 5. 实际小例子：切换显示登录/登出按钮

```jsx
import { useState } from "react";

function App() {
  const [isLoggedIn, setIsLoggedIn] = useState(false);

  return (
    <div>
      <h1>My Website</h1>
      {isLoggedIn ? (
        <div>
          <p>Welcome, User!</p>
          <button onClick={() => setIsLoggedIn(false)}>Logout</button>
        </div>
      ) : (
        <div>
          <p>Please log in</p>
          <button onClick={() => setIsLoggedIn(true)}>Login</button>
        </div>
      )}
    </div>
  );
}

export default App;
```

👉 在这个例子里：

- 如果 `isLoggedIn === true` → 显示 **Welcome, User!** 和 Logout 按钮
    
- 如果 `isLoggedIn === false` → 显示 **Please log in** 和 Login 按钮

---

# ✅ JavaScript Truthy / Falsy 速查表

## 🚫 **Falsy 值（只有这 7 种）**

在布尔判断里会当作 `false`：

|值|说明|示例三元运算符|
|---|---|---|
|`false`|布尔假|`false ? "A" : "B"` → `"B"`|
|`0` / `-0`|数字零|`0 ? "A" : "B"` → `"B"`|
|`0n`|BigInt 零|`0n ? "A" : "B"` → `"B"`|
|`""`|空字符串|`"" ? "A" : "B"` → `"B"`|
|`null`|空值|`null ? "A" : "B"` → `"B"`|
|`undefined`|未定义|`undefined ? "A" : "B"` → `"B"`|
|`NaN`|非数字|`NaN ? "A" : "B"` → `"B"`|

---

## ✅ **Truthy 值（其他所有值）**

只要不是上面 7 种，都会当作 `true`：

|值|说明|示例三元运算符|
|---|---|---|
|`true`|布尔真|`true ? "A" : "B"` → `"A"`|
|非零数字|例如 `1, -5, 3.14`|`123 ? "A" : "B"` → `"A"`|
|非空字符串|`"hello"`, `"false"`, `"0"`|`"0" ? "A" : "B"` → `"A"`|
|对象|`{}`, `{a:1}`|`{} ? "A" : "B"` → `"A"`|
|数组|`[]`, `[0]`|`[] ? "A" : "B"` → `"A"`|
|函数|`function() {}`|`(function(){}) ? "A" : "B"` → `"A"`|
|特殊数值|`Infinity`, `-Infinity`|`Infinity ? "A" : "B"` → `"A"`|

---

## ⚡ 小技巧

- `if (value)` 和 `value ? X : Y` 判断逻辑完全一致。
    
- 想更明确区分，可以用：
    
    - `value === undefined`
        
    - `value === null`
        
    - `Number.isNaN(value)`  
        来避免误判。

在 React 中 **重复使用组件但使用不同的数据** 是非常常见的实践，这正是组件化的核心价值之一。下面是具体做法的详细讲解：

---

## ✅ 使用 props 重复使用组件并传入不同数据

你可以通过传递不同的 props 来重复使用相同的组件，达到渲染不同内容的目的。

### 🎯 示例 1：基本组件复用

```jsx
// UserCard.jsx
function UserCard({ name, age }) {
  return (
    <div className="p-4 border rounded mb-2">
      <h2>{name}</h2>
      <p>{age} years old</p>
    </div>
  );
}

export default UserCard;
```

```jsx
// App.jsx
import UserCard from './UserCard';

function App() {
  return (
    <div>
      <UserCard name="Alice" age={25} />
      <UserCard name="Bob" age={30} />
      <UserCard name="Charlie" age={22} />
    </div>
  );
}
```

---

## ✅ 使用 children 插槽来传入不同结构

当组件需要展示“可变的内容结构”，可以通过 `children` 插槽方式进行复用。

### 🎯 示例 2：卡片组件内部内容变化

```jsx
// Card.jsx
function Card({ children }) {
  return (
    <div className="p-4 shadow rounded bg-white">
      {children}
    </div>
  );
}
```

```jsx
// App.jsx
<Card>
  <h2>标题1</h2>
  <p>这里是内容1</p>
</Card>

<Card>
  <h2>标题2</h2>
  <ul>
    <li>列表项A</li>
    <li>列表项B</li>
  </ul>
</Card>
```

---

## ✅ 使用数组和 map 批量渲染组件

如果你有一组数据，可以通过 `.map()` 动态地渲染多个相同的组件。

### 🎯 示例 3：用 map 显示列表

```jsx
const users = [
  { name: 'Tom', age: 28 },
  { name: 'Jerry', age: 21 },
  { name: 'Spike', age: 35 },
];

function App() {
  return (
    <div>
      {users.map((user, index) => (
        <UserCard key={index} name={user.name} age={user.age} />
      ))}
    </div>
  );
}
```

---

## ✅ 提升：传入函数、事件或组件来自定义行为

你甚至可以传递函数或组件本身作为 props，这样复用组件时就可以定义行为和样式。

### 🎯 示例 4：传入 onClick、renderProp 等

```jsx
function Button({ label, onClick }) {
  return <button onClick={onClick}>{label}</button>;
}

function App() {
  const sayHi = () => alert("Hi!");
  const sayBye = () => alert("Bye!");

  return (
    <>
      <Button label="Greet" onClick={sayHi} />
      <Button label="Farewell" onClick={sayBye} />
    </>
  );
}
```

---

## 小结

|技术|用法|场景|
|---|---|---|
|`props`|传数据|同结构不同内容|
|`children`|插槽|自定义内容结构|
|`.map()`|批量渲染|动态列表|
|函数/组件作为 props|行为控制|更高级复用场景|

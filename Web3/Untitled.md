## **Day 1：Web3 基础 & 区块链概念**

**目标**：理解区块链、智能合约、Web3 架构

1. **核心概念**
    
    - 区块链原理：块、链、节点、共识机制（PoW/PoS）
        
    - 加密货币基础：钱包、私钥、公钥、交易
        
    - 智能合约：什么是合约，为什么它不可篡改
        
    - Web3 与 Web2 区别：去中心化 vs 中心化
        
2. **动手练习**
    
    - 安装 MetaMask 钱包（Chrome 插件）
        
    - 创建测试钱包，连接测试网（如 Ropsten / Goerli）
        
    - 查看区块链浏览器（Etherscan）
        

**推荐教程**

- [Buildspace Web3 Basics](https://buildspace.so/)
    
- [CryptoZombies 教学](https://cryptozombies.io/) — Solidity 入门
    

---

## **Day 2：Solidity & 智能合约开发**

**目标**：写一个简单智能合约，部署到测试网

1. **Solidity 基础**
    
    - 数据类型：uint, string, address, bool
        
    - 函数、条件语句、循环
        
    - 事件（Events）、modifier
        
    - ERC-20 / ERC-721 标准理解
        
2. **开发工具**
    
    - Remix IDE（在线 Solidity 编辑器）
        
    - Hardhat / Truffle（本地开发、测试框架）
        
    - Ganache（本地测试链）
        
3. **动手练习**
    
    - 用 Remix 写一个简单的 “Hello Web3” 合约（存储字符串）
        
    - 部署到 Goerli 测试网
        
    - 使用 MetaMask 调用合约函数
        

**推荐教程**

- [Solidity by Example](https://solidity-by-example.org/)
    
- [Remix IDE Tutorial](https://remix.ethereum.org/)
    

---

## **Day 3：Web3 前端集成**

**目标**：让 Web 页面和智能合约交互

1. **核心技术**
    
    - JavaScript / TypeScript 基础
        
    - web3.js 或 ethers.js（和区块链交互的库）
        
    - 连接 MetaMask 钱包
        
    - 调用智能合约函数
        
2. **动手练习**
    
    - 创建一个 HTML + JS 页面
        
    - 通过 ethers.js 读取和写入智能合约数据
        
    - 展示交易状态、钱包地址
        
3. **进阶方向**
    
    - 使用 React + Next.js + web3.js 构建 DApp
        
    - 了解去中心化存储（IPFS、Filecoin）
        
    - 学习 Layer2 / Rollup 技术
        

**推荐教程**

- [Web3.js Documentation](https://web3js.readthedocs.io/)
    
- [Ethers.js Documentation](https://docs.ethers.io/)
    
- [Buildspace First Web3 App](https://buildspace.so/p/first-web3-app)
    

---

### 🔑 小贴士

- 3 天时间只能做 **动手实验 + 小项目**，不要追求深入理论
    
- 最快上手方式：**Remix + MetaMask + 测试网部署一个简单合约**
    
- 一旦熟悉基础，再考虑 **React DApp + ethers.js** 集成

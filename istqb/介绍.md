ISTQB Foundation Level（全称：**International Software Testing Qualifications Board Certified Tester Foundation Level**，简称 CTFL）是全球公认的软件测试基础认证，适合刚进入测试行业或希望系统学习软件测试理论与实践的人员。

---

## 🧩 一、ISTQB Foundation Level 是什么？

ISTQB Foundation Level 是 ISTQB（国际软件测试认证委员会）体系中的**入门级认证**，为后续的高级和专家级认证打下基础。它涵盖了**软件测试的基本概念、术语、测试设计方法、测试管理、工具支持等**知识。

✅ 适合以下人群：

- 刚进入软件测试领域的新人；
    
- 有经验但未接受过系统训练的测试人员；
    
- 开发人员、项目经理、QA工程师、业务分析师、IT学生等想了解测试基础知识的人。
    

---

## 📚 二、考试内容结构（2023 最新 Syllabus）

考试内容大致分为以下 **6 大章节**：

|章节|内容简述|
|---|---|
|1. **测试基础（Fundamentals of Testing）**|为什么需要测试、测试的目标、测试与开发的关系、七大测试原则、测试流程等基础知识。|
|2. **软件开发生命周期中的测试（Testing Throughout the Software Development Lifecycle）**|了解测试在不同开发模型中的位置（如瀑布、敏捷）、测试类型（单元、集成、系统、验收）等。|
|3. **静态测试（Static Testing）**|静态测试和动态测试的区别，评审和静态分析的流程与好处。|
|4. **测试设计技术（Test Techniques）**|各类设计方法，如黑盒（等价类划分、边界值分析）、白盒（语句覆盖、路径覆盖）、经验法等。|
|5. **测试管理（Test Management）**|风险导向测试、测试计划、估算、监控、控制、配置管理等管理相关内容。|
|6. **测试工具（Tool Support for Testing）**|各类测试工具的分类与用途（测试管理、静态分析、自动化测试等）。|

## 🎯 三、考试形式

|项目|详情|
|---|---|
|📝 题型|40 道 **单项选择题**（四选一）|
|⏱️ 时长|60 分钟（非英语母语者可加时 25%）|
|🎯 通过标准|至少得分 65%（即**26题正确**）|
|💬 语言|英文为主，部分地区提供中文、德文、法文等语言版本|
|🪪 形式|线下考试中心 或 在线远程考试（Remote Proctored）|

---

## 🧠 四、考试难度与准备建议

**难度：中等偏低**，更偏重理解和应用，不涉及编程，但需掌握术语。

### 📘 推荐备考材料：

1. **官方 Syllabus**（最重要，考试内容全部基于此）：
    
    - [https://www.istqb.org](https://www.istqb.org/)
        
2. **Glossary 术语词典**：
    
    - 用于理解专业词汇，例如 test oracle、regression testing 等。
        
3. **练习题库**：
    
    - 网上有大量免费的模拟题（也可参考 Udemy、Simplilearn 等课程）
        
4. **推荐书籍**：
    
    - _Foundations of Software Testing – ISTQB Certification_（作者：Rex Black 等）
        
    - 中文翻译版本也广为流传

## 📈 六、证书有效性

- **证书终身有效**，无需续期；
    
- 可在 ISTQB 官网查询证书编号和验证；
    
- 对申请测试职位、升职、面试有加分作用（全球认可）；
    
- 是进入 **高级级别（Advanced Level）** 的前置要求。
    

---

## 🛠 七、认证机构

在澳洲，ISTQB 是由 **ANZTB（澳新测试委员会）** 负责运营的分支，你可以在以下机构报名考试：

- [ANZTB 官网](https://www.anztb.org/)
    
- [BCS](https://www.bcs.org/)（英国计算机学会，也提供国际报名）
    
- Pearson VUE（考点安排）
    

---

## 🧭 八、进阶路径（未来可以考）

通过 Foundation Level 后，你可以选择以下高级认证路径：

- **Advanced Level**：
    
    - Test Analyst
        
    - Test Manager
        
    - Technical Test Analyst
        
- **Specialist 级别**：
    
    - Agile Testing
        
    - Test Automation Engineer
        
    - Performance Testing 等

## Document Intelligence
- **概述**
  - 定义：自动提取、理解和处理文档中的信息的能力
  - 应用场景：合同审核、发票处理、表单识别等

- **基本概念**
  - 文档类型：结构化、半结构化、非结构化文档
  - 数据提取技术：OCR、NLP、机器学习

- **学习内容**
  - **OCR（光学字符识别）**
    - OCR 的基本原理
    - Azure Cognitive Services 中的 OCR API
    - 应用案例：扫描文档转文本

  - **自然语言处理（NLP）**
    - NLP 的基本概念
    - 文本预处理和分词
    - 情感分析和关键词提取
    - 应用案例：合同中的关键信息提取

  - **机器学习与模型训练**
    - 机器学习基础知识
    - 训练自定义模型的流程
    - 数据标注和清洗
    - 应用案例：文档分类和内容推荐

  - **Azure Document Intelligence**
    - 使用 Azure Form Recognizer 提取表单数据
    - 文档理解与分析的 API
    - 自定义模型的创建和使用
    - 应用案例：自动化发票处理系统

- **实施与集成**
  - 设计 Document Intelligence 解决方案的步骤
  - 与现有系统的集成方法
  - 监控和优化 Document Intelligence 应用
  - 应用案例：客户支持文档的智能搜索系统

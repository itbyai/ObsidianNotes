Microsoft Azure 提供了多种 Azure AI 服务，可用于检测和分析面部，其中包括：

- **Azure AI Vision**，提供面部检测和一些基本的面部分析功能，例如返回图像周围的边界框坐标。
- **Azure AI Video Indexer**，可用于检测和识别视频中的人脸。
- **Azure AI Face**提供可检测、识别和分析面部的预构建算法。

其中，Face 提供了最广泛的面部分析功能。

## Azure AI Face 服务

Azure AI Face 服务可以返回图像中任何人脸的矩形坐标，以及一系列相关属性：

- **配饰**：表示给定的脸部是否有配饰。此属性返回可能的配饰，包括头饰、眼镜和口罩，每个配饰的置信度得分介于 0 和 1 之间。
- **模糊**：脸部模糊的程度，可以表明脸部成为图像主要焦点的可能性。
- **曝光**：例如图像是否曝光不足或曝光过度。这适用于图像中的脸部，而不是整体图像曝光。
- **眼镜**：此人是否戴眼镜。
- **头部姿势**：面部在 3D 空间中的方向。
- **Mask**：表示脸部是否戴着口罩。
- **噪点**：指图像中的视觉噪点。如果您在较暗的环境中使用高 ISO 设置拍摄照片，您会注意到图像中的这种噪点。图像看起来颗粒状或充满微小的点，使图像不太清晰。
- **遮挡**：确定图像中是否有物体遮挡脸部。
- **识别质量**：高、中、低的评级，反映图像的质量是否足以进行人脸识别。

## 负责任地使用人工智能

 重要的

为了支持微软的[负责任的人工智能标准](https://blogs.microsoft.com/on-the-issues/2022/06/21/microsofts-framework-for-building-ai-systems-responsibly/)，Azure AI Face 和 Azure AI Vision 有一个[有限访问策略](https://aka.ms/AAh91ff)。

任何人都可以使用人脸识别服务来执行以下操作：

- 检测图像中人脸的位置。
- 确定一个人是否戴眼镜。
- 确定面部是否存在遮挡、模糊、噪点或过度/曝光不足的情况。
- 返回图像中每个脸部的头部姿势坐标。

有限访问政策要求客户[提交入职表格](https://aka.ms/facerecognition)以访问其他 Azure AI Face 服务功能，包括：

- 人脸验证：比较人脸的相似性的能力。
- 人脸识别：识别图像中指定个人的能力。
- 活体检测：检测并缓解重复出现的内容和/或违反政策的行为的能力（例如输入视频流是真实的还是假的）。

## 用于人脸的 Azure 资源

若要使用人脸服务，必须在 Azure 订阅中创建以下类型的资源之一：To use the Face service, you must create one of the following types of resource in your Azure subscription:

- **人脸**：如果你不打算使用任何其他 Azure AI 服务，或者想要单独跟踪人脸的利用率和成本，请使用此特定资源类型。
- **Azure AI 服务**：一种常规资源，包括 Azure AI Face 以及许多其他 Azure AI 服务（例如 Azure AI 内容安全、Azure AI 语言等）。如果您计划使用多个 Azure AI 服务并希望简化管理和开发，请使用此资源类型。

## 获得更准确结果的技巧

有一些考虑因素可以帮助提高图像检测的准确性：

- 图像格式-支持的图像为 JPEG、PNG、GIF 和 BMP。
- 文件大小 - 6 MB 或更小。
- 脸部尺寸范围 - 从 36 x 36 像素到 4096 x 4096 像素。小于或大于该尺寸的脸部将不会被检测到。
- 其他问题 - 极端的面部角度、极端的灯光和遮挡（挡住面部的物体，例如手）可能会影响面部检测。

"Files path" 指的是网站上文件的路径，即文件在服务器上的位置。网站的文件路径可以分为两种：绝对路径和相对路径。

1. **绝对路径**：
   - 绝对路径是文件在服务器上的完整路径，从根目录开始一直到文件的位置。例如：`/var/www/html/images/picture.jpg`。这种路径是完全指定文件位置的，不受当前页面位置的影响。

2. **相对路径**：
   - 相对路径是相对于当前页面的位置来指定文件的路径。相对路径不包含根目录，而是从当前页面所在的文件夹开始。相对路径分为两种类型：相对于当前文件夹的路径和相对于根目录的路径。
   - 相对于当前文件夹的路径示例：`images/picture.jpg`。这表示 `picture.jpg` 文件位于当前页面所在文件夹内的 `images` 文件夹中。
   - 相对于根目录的路径示例：`/images/picture.jpg`。这表示 `picture.jpg` 文件位于网站的根目录下的 `images` 文件夹中。

在编写网页时，通常使用相对路径，因为它们更加灵活，页面移动到其他文件夹时不需要修改路径。但在某些情况下，如引用其他网站上的文件或在不同页面间共享资源时，可能需要使用绝对路径。

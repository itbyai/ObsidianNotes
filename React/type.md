在编程中，"type alias"（类型别名）是一种为现有类型创建新名称的方式。通过使用类型别名，你可以使代码更具可读性，尤其是在处理复杂类型时。类型别名并不创建新的类型，只是为现有类型提供一个新的引用。

例如，在 TypeScript 中，你可以使用 `type` 关键字定义一个类型别名：

```typescript
type Point = {
  x: number;
  y: number;
};

const p: Point = { x: 10, y: 20 };
```

在这个例子中，`Point` 是一个类型别名，它代表一个包含 `x` 和 `y` 属性的对象。这样使用可以让代码更清晰，并且可以在多个地方重用这个别名。 

不同的编程语言可能有不同的语法和实现，但类型别名的核心思想是相同的。

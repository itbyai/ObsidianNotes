当谈论 CSS 属性时，我们通常指的是样式表中用于控制网页布局和外观的各种属性。以下是一些常见的 CSS 属性以及它们的作用和用法：

1. **color**: 用于设置文本颜色。可以使用颜色名称、十六进制值或 RGB 值来指定颜色。参考网站：colorthunt.co 

   ```css
   color: red; /* 使用颜色名称 */
   color: #ff0000; /* 使用十六进制值 */
   color: rgb(255, 0, 0); /* 使用 RGB 值 */
   ```

2. **font-size**: 设置字体大小。可以使用像素、百分比或其他长度单位来指定大小。

   ```css
   font-size: 16px; /* 使用像素单位 */
   font-size: 100%; /* 使用百分比 */
   ```

3. **font-family**: 设置字体系列。指定使用哪种字体来显示文本。

   ```css
   font-family: Arial, sans-serif; /* 指定优先使用 Arial 字体，如果无法显示，则使用 sans-serif 作为备选字体 */
   ```

4. **font-weight**: 设置字体的粗细。通常使用关键字或数值来指定。

   ```css
   font-weight: bold; /* 设置为粗体 */
   ```

5. **text-align**: 控制文本水平对齐方式。

   ```css
   text-align: center; /* 文本居中对齐 */
   ```

6. **line-height**: 设置行高，即行与行之间的垂直间距。

   ```css
   line-height: 1.5; /* 设置为当前字体大小的 1.5 倍 */
   ```

7. **background-color**: 设置元素的背景颜色。

   ```css
   background-color: #f0f0f0; /* 设置背景色为浅灰色 */
   ```

8. **margin**: 设置元素的外边距。

   ```css
   margin: 10px; /* 上下左右外边距均为 10 像素 */
   ```

9. **padding**: 设置元素的内边距。

   ```css
   padding: 5px; /* 上下左右内边距均为 5 像素 */
   ```

10. **border**: 设置元素的边框。

    ```css
    border: 1px solid #ccc; /* 1 像素宽的实线边框，颜色为浅灰色 */
    ```

这些只是 CSS 属性的一小部分，CSS 还包括很多其他属性，用于控制元素的尺寸、位置、动画等等。CSS 的灵活性使得可以根据需要轻松地设计出各种不同风格和布局的网页。

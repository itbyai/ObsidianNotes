# 机器学习学习路径与资料

要成为 **机器学习专家**，建议分为四个阶段：基础 → 算法 → 工程化 → 专业方向。下面是逐步学习路线与推荐资源。

---

## 阶段一：数学与编程基础
**目标**：掌握机器学习所需的数学与编程能力。

- 数学：
  - 线性代数（矩阵运算、特征值分解）  
  - 概率与统计（贝叶斯定理、分布、假设检验）  
  - 微积分（梯度、偏导、最优化）  
- 编程：
  - Python（Numpy、Pandas、Matplotlib、Scikit-learn）  
  - 数据处理与可视化  

**推荐资料**：
- [3Blue1Brown: 线性代数本质](https://www.3blue1brown.com/)  
- 《统计学习方法》（李航）  
- 《Mathematics for Machine Learning》（Marc Deisenroth）  
- Python 入门: [Real Python](https://realpython.com/)  

---

## 阶段二：机器学习核心算法
**目标**：理解经典机器学习算法原理与实践。

- 有监督学习：线性回归、逻辑回归、SVM、决策树、随机森林、梯度提升  
- 无监督学习：聚类（K-means、DBSCAN）、降维（PCA、t-SNE）  
- 模型评估：交叉验证、混淆矩阵、ROC/AUC  

**推荐资料**：
- 《Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow》（Aurélien Géron）  
- [Coursera - Andrew Ng: Machine Learning](https://www.coursera.org/learn/machine-learning)  
- [Kaggle Learn](https://www.kaggle.com/learn)  

---

## 阶段三：深度学习与工程化
**目标**：掌握深度学习与大规模工程应用。

- 深度学习：
  - 神经网络（MLP、CNN、RNN、Transformer）  
  - 框架：TensorFlow、PyTorch  
- 工程化：
  - ML Ops（模型部署、CI/CD、监控）  
  - 数据管道（ETL、数据清洗、特征工程）  
- 应用：
  - 自然语言处理（NLP）  
  - 计算机视觉（CV）  
  - 推荐系统  

**推荐资料**：
- [Deep Learning Specialization (Andrew Ng, Coursera)](https://www.coursera.org/specializations/deep-learning)  
- 《Deep Learning》（Ian Goodfellow, 花书）  
- [Fast.ai Practical Deep Learning](https://course.fast.ai/)  
- [Made With ML (MLOps 指南)](https://madewithml.com/)  

---

## 阶段四：进阶与专业化
**目标**：深入某一方向，具备研究与创新能力。

- 研究论文阅读：ArXiv、ACL、NeurIPS、ICML  
- 专业方向：
  - **NLP**: Transformer、LLM、RAG、微调  
  - **CV**: 图像生成、目标检测  
  - **强化学习**: AlphaGo, PPO, DQN  
  - **AutoML & AI Agents**: 自动化建模、Agent 系统  

**推荐资料**：
- [Papers With Code](https://paperswithcode.com/)  
- HuggingFace 课程: [https://huggingface.co/learn](https://huggingface.co/learn)  
- 强化学习: 《Reinforcement Learning: An Introduction》（Sutton & Barto）  

---

## 学习方法建议
1. **动手实践**：参与 [Kaggle](https://www.kaggle.com/) 比赛或 GitHub 开源项目。  
2. **项目驱动**：构建自己的 ML 项目（如情感分析、推荐系统、图像识别）。  
3. **持续学习**：订阅 ArXiv、Medium AI 专栏、Twitter/LinkedIn 上的 AI 专家。  
4. **系统化复盘**：每完成一个项目，总结技术点与改进点。  

---

## 下一步行动
- 你已有 **Python/C#/TypeScript** 背景，建议先快速补齐数学 & ML 基础 → 然后进入 **深度学习框架（PyTorch）** → 最后结合 **AI Agents 和 MLOps**。  
- 可先跟着 **Andrew Ng 的 Machine Learning + Deep Learning Specialization**，再进入 Fast.ai 实战。



# IBKR "Refer-A-Friend" Program (Terms & Conditions, 9 Sep 2025) 要点总结

## 奖励机制
- **被推荐人 (Referred Client)**  
  - 每存入 **$300 资金/资产净额**，获赠 **价值 $1 的 IBKR A 类普通股**。  
  - 奖励上限：**$1,000 股票**。  
  - 只能参加一次该活动。  
  - 股票锁定期：**1 年内不可出售/转让**，否则作废。  
  - 奖励股票会随存取款动态增减，最终以 1 年期满时净存额计算。

- **推荐人 (Referring Client)**  
  - 每成功推荐 1 位合格客户，可获得 **$200 固定现金奖励**。  
  - 条件：被推荐人开户、30 天内存入至少 $10,000 并维持 1 年。  
  - 每年最多 15 人，总上限 30 人。  
  - 同一被推荐人无论开多个账户，只算一次奖励。

## 资格限制
- **不可参加群体**：  
  - 财务顾问、对冲基金、介绍经纪商、自营交易公司。  
  - 西班牙、葡萄牙、日本、丹麦、以色列、波兰居民。  
  - IBKR 加拿大、印度开户客户（作为被推荐人）。  
  - 税收优惠账户（如退休账户）。  
  - IBKR 员工。  
  - 与推荐人同住家庭成员。  

- **推荐人资格**：  
  - 不得作为 IBKR 员工/代理。  
  - 不得协助开户、下单、管理或培训被推荐人账户。  
  - 不得充当财务顾问或代理。

## 股票规则
- 股票发放会随存入和取出动态变化。  
- 示例：  
  - 存入 $30,000（股价 $50）：奖励 2 股。  
  - 若次日取出 $15,000，则收回 1 股。  
- 锁定期 1 年，不可出售/转让。  
- 可领取分红和投票权。  
- 若账户关闭/违规，未归属的股票将被没收。  

## 税务责任
- 参与者需自行承担相关税务申报和缴纳。  
- IBKR 可代扣或强制出售股票缴税。  
- 美国税务居民：股票解禁时的市值视为普通收入，并出具 **1099-MISC**。  

## 其他条款
- IBKR 可随时修改或终止活动。  
- 法律管辖：纽约州法律。  
- 争议解决：强制仲裁，放弃陪审团审判权。  
- 协议条款若部分无效，其余仍有效。  
- 本计划不构成证券推荐或招揽。

## 高频易混淆概念

### Guardrail

- 安全约束
- 不是 Hallucination

### Hallucination

- 模型编造
- 缺乏依据

### Ground Truth vs Golden Dataset

- Ground Truth = 单条真实答案
- Golden Dataset = 一组高质量评估样本

### Inference Logging vs Inference Tables vs Tracing

- Logging = 记录推理
- Tables = 存日志的表
- Tracing = 看执行过程

### ETL vs ELT

- ETL = 先转后载
- ELT = 先载后转

### Job vs Pipeline

- Job = 调度编排
- Pipeline = 数据管道定义

### Similarity Search vs Hybrid Search

- Similarity = 纯向量近邻
- Hybrid = 向量 + 关键词 / 过滤

### Precision vs Recall

- Precision = 前 K 里有多少相关
- Recall = 该找回的找回了多少

### Evaluation vs Monitoring

- Evaluation = 上线前
- Monitoring = 上线后

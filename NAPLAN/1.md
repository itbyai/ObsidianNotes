# 六、语法知识点

### 名词以 o 结尾的复数变化

- 非生命物体 → 加 **s**
    
    - photo → photos
        
- 生物或特定词 → 加 **es**
    
    - tomato → tomatoes
        
    - potato → potatoes
        
    - tornado → tornadoes
        
    - volcano → volcanoes / volcanos
        
    - echo → echoes

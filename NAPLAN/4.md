# 四、语法知识点

## 1️⃣ 名词单数变复数规则

### （1）一般加 -es

- bus → buses
    
- fox → foxes
    
- dish → dishes
    
- church → churches
    

### （2）f / fe 变 v + es

- wolf → wolves
    
- knife → knives
    
- thief → thieves
    

### 例外（直接加 s）

- chiefs
    
- chefs
    
- beliefs
    
- giraffes
    

### （3）o 结尾

- hero → heroes
    
- volcano → volcanoes / volcanos
    

### （4）单复数相同

- fish
    
- deer
    
- sheep
    

### （5）不规则复数

- man → men
    
- woman → women
    
- child → children
    
- mouse → mice
    
- goose → geese
    
- tooth → teeth
    
- ox → oxen

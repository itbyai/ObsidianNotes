举个例子
一个完整的策略应该回答五个问题：

| 维度      | 问题         | 举例                |
| ------- | ---------- | ----------------- |
| 🎯 目标   | 你想达成什么？    | 稳定月收益5%，或每笔亏损不超2% |
| 🧠 逻辑   | 为什么这策略能赚钱？ | 趋势延续、均值回归、资金面驱动   |
| 📊 入场条件 | 什么时候买？     | 当50日均线向上突破200日均线时 |
| 🕒 出场条件 | 什么时候卖？     | 当价格跌破20日均线或止损-5%时 |
| 💰 资金管理 | 每次买多少？     | 单笔仓位不超过总资金的10%    |

左侧交易 - 长期投资 - 1年以上
研究股票的价值，以及当前的价格是否反映了全部的价值，如果没有，那么买入。

右侧交易 - 短期投资 - 几个月左右
使用均线20日均线和200日均线，上插形成黄金交叉点
使用当前价格与20日均线下插，形成死亡交叉

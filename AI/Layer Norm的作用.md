Layer normalization（层归一化）是一种常用的神经网络正则化技术，它通常用于深度神经网络中的每一层。其作用是对每个样本的每个特征维度进行归一化，以减少不同特征之间的差异，有助于提高模型的训练速度和稳定性。

### 作用

1. **减少 Internal Covariate Shift**：
   - 在训练深度神经网络时，由于每一层的参数更新，导致网络中间层输入分布的变化，称为 Internal Covariate Shift。Layer normalization 可以减少这种分布变化，使得每一层的输入分布更加稳定，有助于加快模型的收敛速度。

2. **降低梯度消失或梯度爆炸**：
   - Layer normalization 可以缓解梯度消失或梯度爆炸的问题，使得梯度在反向传播过程中更加稳定，有助于提高模型的训练效果和收敛速度。

3. **加速训练速度**：
   - 由于归一化每个样本的特征维度，使得神经网络在训练过程中更容易学习到数据的分布特征，从而提高了模型的训练速度。

### 输出结果的特征

Layer normalization 的输出结果具有以下特征：

1. **每个样本的每个特征维度都归一化**：
   - 对每个样本的每个特征维度进行归一化处理，使得输出的每个特征维度的均值接近于0，标准差接近于1。

2. **每个特征维度独立归一化**：
   - 与 Batch normalization 不同，Layer normalization 是对每个样本的每个特征维度进行归一化处理，而不是对每个批次的每个特征维度进行归一化处理。因此，每个特征维度的归一化是独立进行的，不受批次大小的影响。

3. **保留特征间的相对关系**：
   - Layer normalization 不改变特征维度之间的相对关系，只是对每个特征维度进行了线性变换，因此保留了输入特征之间的相关性和分布特征。

4. **输出结果的分布稳定**：
   - Layer normalization 使得每个样本的每个特征维度的分布更加稳定，有助于提高模型的训练速度和稳定性。

总的来说，Layer normalization 通过对每个样本的每个特征维度进行归一化处理，使得输出结果具有更加稳定的分布特征，从而提高了模型的训练效果和收敛速度。

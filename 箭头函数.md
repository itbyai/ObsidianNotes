箭头函数是 JavaScript 中的一种简洁的函数表达式语法，使用 `=>` 符号定义。它具有更简短的语法，并且不会绑定自己的 `this`，通常在回调函数或内联函数中使用。

### 基本语法

```javascript
(param1, param2, ...) => expression
```

### 示例用法

1. **无参数**

```javascript
const sayHello = () => console.log('Hello, World!');
sayHello(); // 输出: Hello, World!
```

2. **单个参数（可以省略括号）**

```javascript
const square = x => x * x;
console.log(square(5)); // 输出: 25
```

3. **多个参数（必须加括号）**

```javascript
const add = (a, b) => a + b;
console.log(add(3, 7)); // 输出: 10
```

4. **多行函数体（需要用 `{}` 并显式 `return`）**

```javascript
const multiply = (a, b) => {
    const result = a * b;
    return result;
};
console.log(multiply(3, 7)); // 输出: 21
```

5. **箭头函数和数组方法**

```javascript
const numbers = [1, 2, 3];
const doubled = numbers.map(n => n * 2);
console.log(doubled); // 输出: [2, 4, 6]
```

### 特性

1. **没有自己的 `this`**
    
    - 箭头函数会继承定义它时所在上下文的 `this`，不像普通函数会动态绑定 `this`。
2. **不能作为构造函数**
    
    - 箭头函数不能使用 `new` 关键字实例化对象。
3. **没有 `arguments` 对象**
    
    - 可以使用 `...rest` 参数来代替。

### 对比普通函数

```javascript
function normalFunction() {
    return 'This is a normal function';
}

const arrowFunction = () => 'This is an arrow function';
```

使用箭头函数可以使代码更简洁，但在某些需要特定 `this` 绑定的场景中，应优先使用普通函数。

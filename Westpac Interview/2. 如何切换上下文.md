---

## ✅ **问题背景**

混合应用（hybrid app）既包含：

- Native 元素（原生 UI）
    
- WebView 元素（嵌入的 HTML 页面）
    

Appium 的原理：需要告诉驱动切换到哪个“上下文”（context）才能找到对应的元素。

---

## ⚙ **具体做法步骤**

### **1️⃣ 启动 App 并初始化 driver**

正常创建 Appium driver 实例：

```python
from appium import webdriver

desired_caps = {
    'platformName': 'Android',
    'deviceName': 'emulator-5554',
    'appPackage': 'com.example.app',
    'appActivity': '.MainActivity',
    'automationName': 'UiAutomator2',
    'autoWebview': False   # 不要自动切换，手动控制
}
driver = webdriver.Remote('http://localhost:4723/wd/hub', desired_caps)
```

---

### **2️⃣ 获取当前上下文**

```python
print(driver.current_context)  
# 输出通常是：'NATIVE_APP'
```

---

### **3️⃣ 获取所有可用上下文**

```python
contexts = driver.contexts
print(contexts)
# 示例输出：
# ['NATIVE_APP', 'WEBVIEW_com.example.app']
```

> - `NATIVE_APP`: 原生上下文
>     
> - `WEBVIEW_...`: 应用内的 WebView 上下文  
>     WebView 上下文名称取决于包名或具体配置。
>     

---

### **4️⃣ 切换到 WebView**

```python
driver.switch_to.context('WEBVIEW_com.example.app')
```

> 切换后才能通过常规 Web 定位方式（xpath、css selector 等）访问 WebView 里的 HTML 元素。

---

### **5️⃣ 在 WebView 中进行自动化操作**

```python
element = driver.find_element_by_css_selector('button.submit')
element.click()
```

---

### **6️⃣ 切换回 Native**

```python
driver.switch_to.context('NATIVE_APP')
```

> 回到 native 后才能再次操作原生元素（比如系统弹窗、tabbar 等）。

---

## 🛠 **实用技巧**

✅ WebView 有时不会立刻出现：

- 可以循环检查，直到 `driver.contexts` 返回包含 `WEBVIEW_...`。
    

```python
import time

timeout = 10
for _ in range(timeout):
    contexts = driver.contexts
    if len(contexts) > 1:
        break
    time.sleep(1)
driver.switch_to.context(contexts[-1])  # 切到最后一个
```

---

✅ 有时需要先开启 WebView 调试：

- Android：开发者选项里**开启 WebView 调试**
    
- iOS：需签名支持 WebView 自动化
    

---

✅ **autoWebview capability**

- 如果加了 `'autoWebview': True`：  
    Appium 会在 session 创建后自动切换到 WebView。
    
- 不建议一开始就用，因为有时需要先做 native 操作（如登录、权限弹窗）。
    

---

## 📌 **总结**

> Appium 测试 hybrid app 的核心是：
> 
> - 获取上下文 `driver.contexts`
>     
> - 切换 `driver.switch_to.context()`
>     
> - 回到原生 `NATIVE_APP`
>     

流程是：  
1️⃣ 启动 App  
2️⃣ 获取上下文列表  
3️⃣ 切换到 WebView → 测试 Web 部分  
4️⃣ 切回 NATIVE_APP → 测试 native 部分

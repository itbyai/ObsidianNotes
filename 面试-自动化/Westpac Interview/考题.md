## 🧩 题目：判断一个整数是否为回文数（Palindrome Number）

### 描述

给定一个整数 `x`，如果 `x` 是一个回文整数，返回 `true`，否则返回 `false`。  
回文整数指正序和倒序读都是一样的数字。

例如：

```
输入: 121
输出: true
解释: 从左到右读是 121，从右到左也是 121。
```

```
输入: -121
输出: false
解释: 从左到右是 -121，从右到左是 121-。
```

```
输入: 10
输出: false
解释: 从右到左是 01。
```

---

### ✅ 解法 1：反转整数法（不转字符串）

```java
public class Solution {
    public boolean isPalindrome(int x) {
        if (x < 0) return false; // 负数不是回文
        int original = x;
        int reversed = 0;

        while (x != 0) {
            int digit = x % 10;
            reversed = reversed * 10 + digit;
            x /= 10;
        }

        return original == reversed;
    }
}
```

**时间复杂度**：O(log₁₀N)  
**空间复杂度**：O(1)

---

### ✅ 解法 2：字符串法（适合初学者）

```java
public boolean isPalindrome(int x) {
    String s = Integer.toString(x);
    int i = 0, j = s.length() - 1;
    while (i < j) {
        if (s.charAt(i++) != s.charAt(j--)) return false;
    }
    return true;
}
```

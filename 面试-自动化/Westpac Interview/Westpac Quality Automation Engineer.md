以下是 **Westpac Quality Automation Engineer** 职位的要点总结 👇

---

### 🏦 **职位名称**

**Quality Automation Engineer**  
地点：Surfers Paradise, QLD, Australia  
所属项目：**UNITE 转型计划（业务驱动 + 技术赋能）**

---

### 🎯 **职位概述**

- 作为 Westpac **UNITE 转型项目**核心团队成员，负责通过自动化测试提高银行关键平台的质量和交付效率。
    
- 目标是将手动流程标准化、自动化，简化系统与技术，实现更佳客户与员工体验。
    

---

### ⚙️ **主要职责**

- 将手动回归测试转化为自动化测试套件。
- 使用 **Playwright** 设计、开发、维护自动化脚本。
- 使用 **RestAssured** 进行 **API 测试** 与微服务验证。
- 与 **敏捷团队（agile squads）** 合作，定义测试范围与质量标准。
- 在 **CI/CD 流水线**（Azure DevOps / Jenkins）中集成自动化测试。
- 维护测试环境，确保自动化执行稳定。
- 记录和分析缺陷，支持快速修复。

---

### 🧠 **任职要求**

- 计算机科学或相关专业学位。
    
- **5–7年** 企业级功能与自动化测试经验。
    
- 精通 **Selenium (Java)**、**Playwright** 或 **Cypress**。
    
- 熟练进行 **RESTful API 测试与契约验证**。
    
- 熟悉 **Java / JavaScript / Python** 编程。
    
- 有微服务架构与分布式系统测试经验。
    
- 熟悉 **Agile 工作模式** 和 **CI/CD 工具（Azure DevOps、Jenkins）**。
    

---

### 🌟 **为什么加入 Westpac**

- 参与集团级 **数字化转型项目（UNITE）**，影响深远。
    
- 享受银行产品专属优惠与员工折扣（如住房贷款利率优惠）。
    
- 灵活工作安排（远程办公、弹性时间、额外假期）。
    
- 多样的学习与职业发展机会。
    
- 志愿服务与社区参与机会。
    
- 包容多元的工作环境（支持退伍军人、原住民及神经多样性群体）。

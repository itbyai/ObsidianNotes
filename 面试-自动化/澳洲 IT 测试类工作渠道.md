# 澳洲 IT 测试类工作渠道

澳洲的 IT 测试类岗位（QA Engineer、Test Analyst、Automation Tester 等）可以通过多种渠道寻找，包括招聘网站、猎头、社交网络以及行业活动。

---

## 1. 在线招聘网站

| 网站 | 特点 |
|------|------|
| **Seek** ([seek.com.au](https://www.seek.com.au/)) | 澳洲最大综合招聘平台，可按职位类型、城市筛选 IT 测试岗位 |
| **Indeed** ([indeed.com.au](https://au.indeed.com/)) | 聚合职位信息，支持按公司、薪资和工作经验搜索 |
| **LinkedIn Jobs** ([linkedin.com/jobs](https://www.linkedin.com/jobs/)) | 可直接与招聘方建立联系，适合中高级岗位和外企 |
| **Jora** ([jora.com.au](https://au.jora.com/)) | 多渠道职位聚合，更新频繁 |
| **Adzuna** ([adzuna.com.au](https://www.adzuna.com.au/)) | 提供职位市场分析、薪资参考 |

---

## 2. IT 专业招聘平台

| 网站 | 特点 |
|------|------|
| **Stack Overflow Jobs** ([stackoverflow.com/jobs](https://stackoverflow.com/jobs)) | 面向开发和测试类岗位，部分远程岗位可申请 |
| **GitHub Jobs** ([jobs.github.com](https://jobs.github.com/)) | 技术社区岗位，适合 Automation Tester 或 DevOps 测试岗位 |
| **Australian Computer Society (ACS) Career Centre** ([acs.org.au](https://www.acs.org.au/careers.html)) | 行业协会提供的 IT 岗位信息，含测试岗位和认证相关职位 |

---

## 3. 猎头与招聘公司

| 公司 | 说明 |
|------|------|
| **Hays IT** ([hays.com.au](https://www.hays.com.au/)) | 专注 IT 招聘，覆盖 QA、Automation、Test Analyst |
| **Robert Half Technology** ([roberthalf.com.au](https://www.roberthalf.com.au/)) | 专业 IT 招聘，提供合同和全职岗位 |
| **Michael Page IT** ([michaelpage.com.au](https://www.michaelpage.com.au/)) | 高级技术岗位猎头，适合有经验的测试工程师 |
| **Peoplebank** ([peoplebank.com.au](https://www.peoplebank.com.au/)) | 澳洲本地 IT 人才招聘公司，提供 QA/测试岗位 |

---

## 4. 社交网络与技术社区

- **LinkedIn**：关注澳洲 IT 企业和招聘负责人，直接投递简历。
- **Meetup** ([meetup.com](https://www.meetup.com/))：参加 QA、Testing、Agile、DevOps 社区活动，拓展人脉。
- **Slack / Discord IT 群组**：部分城市有 QA/Automation Tester 社区，发布招聘信息。

---

## 5. 公司官网与实习机会

- 大型 IT 企业和银行通常在官网发布岗位：
  - **Commonwealth Bank, ANZ, NAB, Westpac**：金融行业 QA/测试岗位
  - **Atlassian, Canva, Wise**：科技企业 QA/Automation
- 留意实习或毕业生项目，适合应届生或转行测试。

---

## 6. 其他渠道

- **高校就业中心**：澳洲大学提供毕业生就业辅导和招聘信息。
- **招聘会/行业大会**：参加 IT/软件测试大会，面对面交流。
- **自由职业平台**：如 Upwork、Freelancer，可承接自动化测试或 QA 项目。

---

## 7. 求职建议

1. **简历优化**：突出测试工具（Selenium, Cypress, Postman）、脚本语言（Python, Java, JavaScript）及自动化经验。
2. **认证提升**：ISTQB、Certified Agile Tester 等证书有助于增加竞争力。
3. **网络拓展**：LinkedIn 建立人脉，加入澳洲 QA 群组。
4. **投递策略**：同时使用招聘网站、猎头和公司官网，增加成功率。

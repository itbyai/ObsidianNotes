# 📌 Playwright 简介及其特性

## 1. 什么是 Playwright？

Playwright 是由 **微软开发的开源自动化测试框架**，用于现代 Web 应用的 **端到端（E2E）测试**。  
它可以在不同的 **浏览器、设备和平台** 上进行可靠的测试。

- 2020 年首次发布。
    
- 使用 **TypeScript** 编写，但支持多种语言：**JavaScript、TypeScript、Python、Java 和 .NET**。
    
- 专为解决现代 Web 应用的挑战而设计（单页应用、动态内容、Shadow DOM、iframe 等）。
    

---

## 2. 支持的浏览器和平台

- **Chromium 内核浏览器**（Chrome、Edge）
    
- **Firefox**
    
- **WebKit**（Safari 引擎）
    
- 支持 **Windows、macOS、Linux**
    
- 支持 **移动设备模拟**（Android、iOS）
    

---

## 3. Playwright 的主要特性

### ✅ 跨浏览器自动化

- 一个 API 支持 Chromium、Firefox、WebKit。
    
- 确保测试在不同浏览器上的一致性。
    

### ✅ 多语言支持

- 官方支持：
    
    - **JavaScript / TypeScript**
        
    - **Python**
        
    - **Java**
        
    - **.NET (C#)**
        

### ✅ 自动等待，减少 flaky test

- Playwright 会自动等待元素加载完成（可见、可用、稳定）。
    
- 相比 Selenium，测试用例更稳定。
    

### ✅ 现代 Web 应用支持

- 支持复杂场景：
    
    - 单页应用（SPA）
        
    - Shadow DOM、iFrame
        
    - 网络请求拦截与模拟
        
    - 文件上传/下载
        

### ✅ 高效快速

- 支持 **无头模式**（不显示浏览器 UI），执行速度快。
    
- 支持并行执行、测试分片。
    

### ✅ 内置测试框架

- Playwright Test 自带：
    
    - 测试运行器
        
    - Fixtures（测试环境/数据准备）
        
    - 报告（HTML、JSON、JUnit）
        
    - 并行执行支持
        

### ✅ 网络控制

- 可以拦截和修改网络请求/响应。
    
- 可用于 API Mock、模拟弱网环境。
    

### ✅ 移动设备与权限模拟

- 模拟 iPhone、Pixel、iPad 等设备。
    
- 支持地理位置、相机、麦克风等权限控制。
    

### ✅ 强大的调试工具

- Playwright Inspector（逐步调试工具）。
    
- Trace Viewer（查看测试日志、截图、DOM 快照）。
    

### ✅ 便于 CI/CD 集成

- 与 CI/CD 工具无缝集成（Jenkins、GitHub Actions、GitLab CI、Azure DevOps）。
    

---

## 4. 为什么选择 Playwright？

- **现代化且可靠**，是 Selenium 的替代方案。
    
- **一个 API 跨多个浏览器**。
    
- **减少 flaky test**，提高测试稳定性。
    
- **UI + API + 网络 Mock 一体化**。
    
- 由 **微软支持**，社区活跃。
